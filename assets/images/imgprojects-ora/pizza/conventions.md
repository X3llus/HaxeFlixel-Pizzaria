# Naming conventions

`%modifier%%type%%quantity%`

- Modfidier: The variant of the type. ex. `light` or `dark`
- Type: The single character abbreviation for a type contained in the image
  - `s` = sauce
  - `c` = cheese
  - `m` = mushroom
  - `p` = pepperoni
- Quantity: the densitry/quantity of the type
  - `h` = heavy
  - `l` = light
  - ``(no quantity) = normal
